# 🍕 Pizza Sales Analysis using SQL

## 📌 Project Overview
This project analyzes pizza sales data using SQL to uncover valuable business insights. The analysis focuses on understanding sales performance, customer ordering behavior, and revenue distribution across different pizza categories.

## 📊 Dataset
The dataset contains four tables:
- **orders** – stores order date and time
- **order_details** – stores quantity of pizzas ordered
- **pizzas** – stores pizza size and price
- **pizza_types** – stores pizza name and category

## 🛠 SQL Concepts Used
- Joins
- Aggregations (SUM, COUNT, AVG)
- Subqueries
- Window Functions (RANK)
- GROUP BY & ORDER BY
- Date & Time Functions

## 🔎 Analysis Performed
- Calculated total number of orders placed
- Calculated total revenue generated from pizza sales
- Identified the highest priced pizza
- Determined the most common pizza size ordered
- Found the top 5 most ordered pizzas
- Analyzed category-wise pizza sales
- Determined hourly distribution of orders
- Calculated average pizzas ordered per day
- Identified top pizzas based on revenue
- Calculated percentage contribution of each pizza category
- Analyzed cumulative revenue over time
- Found top pizzas in each category using window functions

## 💡 Key Insights
- A few pizza types contribute significantly to total revenue.
- Certain pizza categories dominate overall sales.
- Order volume peaks during specific hours of the day.
